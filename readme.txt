== Description ==

Import your uploads and favorites videos from Youtube in the easiest way.

You can use a list under Posts menú, Media window or autoimport using a CronJob (Wordpress or Unix).

This plugin is developed by <a href="http://capo.cat/">Pau Capó</a>. Powered by <a href="http://wordpress.menorcadev.com/">WordPress Plugins Development #menorcadev.

== Installation ==

= First time installation =
1. Upload the FOLDER 'wp-autoyoutube' to the /wp-content/plugins/
2. Activate the plugin 'WP-AutoYoutube' through the 'Plugins' menu in admin
3. Go to 'Manage / AutoYoutube' for more instructions

= Upgrade from a previous version =
1. Deactivate 'WP-AutoYoutube' through the 'Plugins' menu in admin
2. Upload the FOLDER 'wp-autoyoutube' to the /wp-content/plugins/
3. Activate the plugin 'WP-AutoYoutube' through the 'Plugins' menu in admin

== Frequently Asked Questions ==

= I need to set my password or some hard configuration? =
No, you just need to know a Youtube username.

= The plugin don't show my favorites videos, what's happen? =
You need to put your favorites videos as public in your Youtube account.
